# pinterestClone
📌 Pinterest Clone
A responsive Pinterest-style landing page built with pure HTML and CSS, featuring a dynamic image grid, signup form.

🚀 Features
Fully Responsive Design – Optimized for mobile, tablet, and desktop screens

Dynamic Image Grid – Pinterest-style masonry layout with CSS Grid and object-fit: cover

Dark Overlay Background – Multiple images with dark overlay for text readability

Signup Form – Clean, centered card with email, password, and birthdate fields

Interactive Navigation – Hamburger menu for mobile devices

Pure HTML/CSS – No JavaScript frameworks or external libraries required

📱 Responsive Breakpoints
Device	Screen Width	Layout Behavior
Mobile	< 768px	Single column, stacked content
Tablet	768px - 1024px	2-3 column grid, side-by-side layout
Desktop	> 1024px	4 column grid, full feature display
🛠️ Technologies Used
HTML5

CSS3 (Flexbox, Grid, Media Queries)

Font Awesome Icons
